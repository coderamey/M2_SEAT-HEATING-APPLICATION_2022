\begin{DoxyEnumerate}
\item THE USER ENTERS INTO CAR AND OCCUPIES THE SEAT.
\item AND THEN HE TURNS ON THE HEAT SENSOR.
\item IF BOTH THE ABOVE CONDITION ARE MEET THE LED WILL TURN ON.
\item THEN THE USER ADJUST THE TEMPERTURE.
\item THE TEMPERATURE VALUE IS DISPLAYED ON THE SCREEN.
\item THEN THE SEAT WILL BE COOLED ACCORDINGLY.
